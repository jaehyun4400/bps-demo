## [2026-04-29] HP 오브 UI 개편 + 혜 귀인 보호막 시각화 + 무기 공격 이펙트 시스템 + GitHub 배포

### UI 개편
- **HP 오브 — 디아블로 스타일 액체 구체로 전면 교체**
  - 기존 도넛 차트(conic-gradient 56px) → Canvas 기반 액체 구체(72px)
  - 파도치는 액체 애니메이션(RAF 루프), 유리 광택 + 입체 림 효과
  - HP 비율에 따른 색상 전환: 빨강(50%↑) / 주황(25%↑) / 진빨강(위험)

### 신규 시스템
- **혜 귀인 보호막 시각화 완료**
  - 오브 내부: 쉴드 수치만큼 HP 액체 위에 하늘색 액체 레이어로 표시
  - 외곽 링: 쉴드 활성 시 파란 펄스 애니메이션
  - 쿨타임 중 흐린 회색 링으로 쿨타임 상태 표시
  - 쉴드 소진 시 링이 확산되며 부서지는 파열(shatter) 이펙트
  - 보스 모드 HP 바에도 파란색 쉴드 구간 추가

- **혜 귀인 보호막 로직 완성**
  - HP 60% 이하 시 최대 HP의 15% 보호막 자동 발동
  - 쉴드 소진 후 20초 쿨타임, 종료 시 HP 60% 이하이면 즉시 재발동
  - 스테이지 진입 즉시 HP 60% 이하 여부 체크 → 즉시 발동
  - 쿨타임은 스테이지 이동 시 초기화

- **무기별 공격 이펙트 시스템**
  - bow: 각도 회전 화살(➤) 발사체 + 파란 링 충격파
  - orb: 파이어볼 발사체 + 빨간 링 충격파
  - 미착용(normal_girl): 👊 주먹 발사체 + 황금 링 충격파
  - 필드·보스 모드 전체 적용, `showRingImpact(x,y,bColor,fColor)` 공용 함수 구현
  - 링 크기 10% 축소 (38px / 23px)

- **화씨벽 범위 공격 재설계**
  - orb 공용 AoE 제거 → orb 무기 기본 단일 타겟으로 변경
  - 화씨벽 전용 `splashAll:0.15` — 매 공격 주 데미지의 15%를 전체 적에게 추가 피해
  - 효과 텍스트: "범위 공격 +25%" → "전체 공격 15%"

### 버그 수정
- **자동돌파 보스 클리어 후 멈춤 수정**
  - D1~D6 보스 스테이지 클리어 시 자동돌파가 다음 던전으로 진행하지 않고 멈추던 문제 수정
  - `isLast` 조건이 "마지막 던전의 마지막 스테이지"일 때만 true가 되도록 교정
- **무기 미착용 시 SD 캐릭터 오표시 수정**
  - 무기를 착용하지 않은 상태에서 `sword_girl.png`가 노출되던 문제 수정
  - `getWpType()` 폴백값 `'sword'` → `''` 로 변경, `normal_girl.png` 정상 표시

### 배포 / 문서
